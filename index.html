Sobre

Olá, sou João Campos, um jovem engenheiro químico em transição para a profissão de desenvolvedor front-end. Com uma sólida formação em engenharia química, descobri um verdadeiro talento na área de tecnologia web. Atualmente, estudo e faço desenvolvimento de sites como freelancer.

Minha jornada no desenvolvimento front-end tem sido inspiradora, e estou sempre em busca de novas oportunidades para aprimorar minhas habilidades e conhecimentos nesse campo em constante evolução. Tenho me dedicado a aprender as linguagens essenciais, como HTML, CSS e JavaScript, e estou animado para expandir meu conhecimento para frameworks e bibliotecas modernas, como TypeScript, React e Angular.

O que me motiva é a possibilidade de criar experiências digitais incríveis e envolventes para os usuários, combinando minha formação em engenharia com minhas habilidades criativas. Tenho uma abordagem analítica e orientada a detalhes, que aprimorei ao longo dos anos trabalhando em projetos complexos como engenheiro químico.

Estou ansioso para colaborar com equipes talentosas, enfrentar desafios empolgantes e contribuir para o desenvolvimento de soluções web de alto nível. Minha dedicação, entusiasmo e habilidades técnicas são a base para o meu sucesso contínuo nesta nova jornada como desenvolvedor front-end.

Hardskills

Possuo um amplo conhecimento nas linguagens HTML5, CSS3, JavaScript e Python além de dominar o Git e GitHub. Pretendo aperfeiçoar cada vez mais meus conhecimentos nessas tecnologias, e expandir meu conhecimento aprendendo TypeScript, React e Angular. Estou aberto também para futuramente aprender tecnologias Backend como PHP, Django e SQL, me tornando assim um desenvolvedor Full Stack.

Softskills

Sou uma pessoa paciente que trabalha bem em equipe, criativo, racional, dedicado a cumprir todas tarefas no prazo estipulado, perfeccionista, atento a detalhes.
